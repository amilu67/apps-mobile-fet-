# Apps Mobile FET - Project Presentation

## Dear FET Community and School Administrators,

I am pleased to introduce **Apps Mobile FET**, a comprehensive suite of three Progressive Web Applications (PWAs) designed to revolutionize how schools manage and distribute timetables generated by FET (Free Timetabling Software).

### ðŸŽ¯ The Problem We're Solving

Schools using FET face recurring challenges:
- **Complex Distribution**: Timetables remain locked in desktop software or exported as static PDFs
- **Real-Time Updates**: Teacher substitutions require manual communication via phone calls, printed notices, or group chats
- **Mobile Access**: Students, parents, and staff cannot easily access schedules from their smartphones
- **Administrative Burden**: Managing daily substitutions and absent classes is time-consuming and error-prone

### ðŸ’¡ Our Solution

Apps Mobile FET bridges the gap between FET's powerful scheduling capabilities and modern mobile accessibility. The system consists of three specialized applications:

#### 1. **Student App** - For Students and Parents
A clean, intuitive mobile app that provides:
- **Instant Access**: View class schedules anywhere, anytime
- **Real-Time Substitutions**: Automatic notifications when teachers are absent
- **Absent Classes**: See which classes are on field trips or events
- **Teacher Search**: Find when and where teachers have lessons
- **Offline Support**: Works even without internet connection after first visit

#### 2. **Staff App** - For School Administration
A powerful management tool featuring:
- **Substitution Management**: Create and publish teacher substitutions in seconds
- **Smart Availability Search**: Automatically find available teachers for any time slot
- **Absent Class Management**: Mark classes as absent (trips, assemblies, etc.)
- **Quick Publishing**: Share substitutions via WhatsApp/Telegram with one click
- **Real-Time Sync**: All changes instantly appear on student devices

#### 3. **Converter App** - FET Integration Tool
The bridge between FET and mobile apps:
- **Direct Import**: Load FET files (.fet) and XML solutions
- **Dynamic Configuration**: Set institution name, school year, and availability codes
- **Automatic Processing**: Extracts classes, teachers, subjects, rooms, and time slots
- **Firebase Upload**: Publishes data to cloud database in one click
- **Statistics Dashboard**: View comprehensive timetable analytics
