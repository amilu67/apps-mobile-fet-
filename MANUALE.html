2. Converter App - Convertitore FET

Questa app permette di convertire i file FET e caricarli su Firebase per renderli disponibili alle altre app.

2.1 Preparazione Files FET

1 Genera l'orario in FET

Completa la generazione dell'orario in FET e assicurati che sia corretto.

2 Esporta la soluzione XML

In FET: File > Export > Simple XML...

Salva il file con nome descrittivo (es: orario_soluzione_2024.xml)

3 Salva il file FET

Salva il file .fet del tuo orario (es: orario_2024.fet)

2.2 Utilizzo del Converter

1 Accedi alla Converter App

Apri l'URL della tua converter app nel browser.

2 Carica il file FET

Clicca su "Scegli file FET" e seleziona il tuo file .fet

3 Carica la soluzione XML

Clicca su "Scegli file soluzione XML" e seleziona il file XML esportato

4 Configura i dati
  • Nome Istituzione: Es: "ITI Augusto Righi"
  • Anno Scolastico: Es: "2024/2025"
  • Materie DisponibilitÃ : Es: "D, Disp" (separa con virgola se hai piÃ¹ codici)
5 Converti e Carica

Clicca su "Converti e Carica su Firebase". Attendi il completamento dell'upload.

ðŸ“Š Statistiche: Dopo la conversione vedrai statistiche dettagliate: numero di classi, docenti, materie, aule e lezioni totali.

5. Consigli e Best Practices

5.1 Aggiornamento Orario

  • Aggiorna l'orario sempre tramite Converter App
  • Testa prima su un progetto Firebase di prova
  • Comunica agli utenti quando aggiorni l'orario
  • Mantieni una copia di backup dei file FET

5.2 Gestione Sostituzioni

  • Inserisci le sostituzioni il prima possibile
  • Verifica sempre il docente sostituto prima di salvare
  • Elimina sostituzioni obsolete o annullate
  • Pubblica in gruppo WhatsApp/Telegram per notifica rapida

5.3 Sicurezza

  • Non condividere le credenziali Firebase
  • Limita l'accesso all'app Staff solo al personale autorizzato
  • Imposta regole corrette nel Firebase Database
  • Monitora l'uso del database dalla Firebase Console

6. Risoluzione Problemi

6.1 L'app non mostra l'orario

Cause possibili:

  • Orario non ancora caricato su Firebase
  • Configurazione Firebase errata
  • Problemi di connessione internet

Soluzione:

  1. Verifica di aver caricato l'orario con la Converter App
  2. Controlla la Firebase Console: Database > Realtime Database
  3. Ricarica la pagina (pull down su mobile)

6.2 Le sostituzioni non si aggiornano

Cause possibili:

  • Cache del browser
  • Service Worker non aggiornato

Soluzione:

  1. Chiudi completamente l'app
  2. Riapri e attendi alcuni secondi
  3. Se persiste: Settings > App > Elimina dati > Reinstalla

6.3 Errore durante la conversione FET

Cause possibili:

  • File FET o XML corrotto
  • Versione FET incompatibile
  • AttivitÃ  senza placement nella soluzione

Soluzione:

  1. Apri la console del browser (F12)
  2. Riprova la conversione e leggi i messaggi di debug
  3. Verifica che tutte le attivitÃ  abbiano un placement nella soluzione FET
  4. Rigenera la soluzione XML da FET

6.4 Docenti non trovati nella ricerca

Causa: Nome docente non esattamente come in FET

Soluzione: Digita solo il cognome, l'autocomplete ti aiuterÃ 

6.5 App non installabile su iPhone

Causa: Devi usare Safari, non Chrome

Soluzione: Apri l'app in Safari e usa "Aggiungi a schermata Home"
